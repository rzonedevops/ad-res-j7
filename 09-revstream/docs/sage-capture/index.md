# Sage Account Takeover & Stock2Shop Reconfiguration (2020-2024)

**Last Updated:** 2026-02-09

This section documents the complete timeline and evidence of the takeover of the RegimA Worldwide Sage Accounting system and the subsequent forced reconfiguration of the Stock2Shop API integration. The evidence, extracted from the RegimA Exchange mailbox, proves a multi-year, premeditated effort by Rynette Farrar to seize control of the company's core financial infrastructure, culminating in a fraudulent ownership transfer in July 2024.

---

## Executive Summary

The evidence shows that **Rynette Farrar** was initially invited as a standard user to the Sage system on **18 May 2020** by the legitimate owner, **Kayla Pretorius**. Following Kayla's death in July 2023, Rynette orchestrated a fraudulent "user switch" with Sage support, culminating on **9 July 2024** when the account ownership was transferred from the deceased Kayla Pretorius to **Peter Faucitt**. This unauthorized switch immediately broke the Stock2Shop API connector, which was linked to Kayla's credentials, halting the flow of e-commerce orders. The subsequent 10-day scramble to reconfigure the API provides a clear audit trail of the technical consequences of the takeover.
