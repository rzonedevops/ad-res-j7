### 6.1 Temporal Proximity Confidence Scoring

**Scoring Framework:**

```scheme
(define (compute-temporal-proximity-confidence days-between-events)
  "Computes confidence score for temporal causation based on days between events"
  (cond
    [(= days-between-events 0) 0.99]  ; Same day = highest confidence
    [(= days-between-events 1) 0.98]  ; 1 day = very high confidence
    [(<= days-between-events 7) 0.95] ; Within 1 week = high confidence
    [(<= days-between-events 30) 0.90] ; Within 1 month = moderate-high confidence
    [(<= days-between-events 60) 0.85] ; Within 2 months = moderate confidence
    [(<= days-between-events 90) 0.75] ; Within 3 months = moderate-low confidence
    [else 0.60]))                      ; Beyond 3 months = low confidence
```

### 6.2 Evidence Strength Scoring

**Scoring Framework:**

```scheme
(define (compute-evidence-strength-score evidence-type documentation-quality)
  "Computes evidence strength score based on type and documentation quality"
  (let* ((type-score (case evidence-type
                       [(documentary) 0.95]
                       [(testimonial) 0.85]
                       [(circumstantial) 0.75]
                       [(expert-opinion) 0.90]
                       [else 0.60]))
         (quality-multiplier (case documentation-quality
                               [(excellent) 1.00]
                               [(good) 0.95]
                               [(adequate) 0.85]
                               [(poor) 0.70]
                               [else 0.50])))
    (* type-score quality-multiplier)))
```

### 6.3 Principle Confidence Scoring

```scheme
(define (compute-principle-confidence-score principle-elements evidence-strength)
```
