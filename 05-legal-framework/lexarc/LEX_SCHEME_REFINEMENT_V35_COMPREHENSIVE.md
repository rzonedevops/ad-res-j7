## Part 4: Evidence-to-Principle Mapping V5

### 4.1 Annexure Strength Scoring Framework

**Scoring Methodology:**
```scheme
(define-annexure-strength-score
  (lambda (annexure-type evidence-quality legal-relevance)
    (let ((base-score (* evidence-quality legal-relevance)))
      (cond
        ((eq? annexure-type 'documentary-evidence)
         (* base-score 1.0))  ; Full weight
        ((eq? annexure-type 'expert-opinion)
         (* base-score 0.95)) ; High weight
        ((eq? annexure-type 'witness-statement)
         (* base-score 0.90)) ; Good weight
        ((eq? annexure-type 'circumstantial-evidence)
         (* base-score 0.80)) ; Moderate weight
        ((eq? annexure-type 'hearsay)
         (* base-score 0.50)) ; Low weight
        (else
         (* base-score 0.60)))))) ; Default moderate-low weight
```

**Evidence Quality Assessment:**
- **0.95-1.00**: Direct documentary evidence (contracts, invoices, bank records)
- **0.85-0.94**: Expert opinions, professional standards documentation
- **0.75-0.84**: Witness statements, correspondence, internal records
- **0.65-0.74**: Circumstantial evidence, timeline analysis
- **0.50-0.64**: Hearsay, secondary sources

**Legal Relevance Assessment:**
- **0.95-1.00**: Directly proves/disproves critical legal element
- **0.85-0.94**: Strongly supports legal principle application
- **0.75-0.84**: Moderately supports legal argument
- **0.65-0.74**: Provides context or background
- **0.50-0.64**: Tangentially relevant
