## Key Concepts

### Invariant Guilt

**Invariant guilt** means proving that an opponent's culpability holds across ALL plausible interpretations of facts, including their own claimed version of events. Mathematically:

```
∀w ∈ W: Guilty(A, w) = True
```

Where W represents all worlds consistent with the evidence.

### The Power of the Framework

Traditional legal argumentation is reactive—each rebuttal opens new argumentative space. The Invariant Guilt Framework is **proactive and exhaustive**, mapping the entire possibility space and proving guilt in every region simultaneously.
