# Case Templates

This directory contains templates for creating new cases. Templates provide standardized structures and boilerplate content for different types of legal analysis.

## Available Templates

### Criminal Case Template
- **Purpose**: Criminal case timeline and evidence analysis
- **Structure**: Follows South African criminal law procedures
- **Includes**: Timeline framework, evidence checklist, documentation templates

### Civil Litigation Template  
- **Purpose**: Civil dispute analysis and case management
- **Structure**: Civil procedure compliance
- **Includes**: Pleading frameworks, discovery tracking, settlement analysis

### Corporate Investigation Template
- **Purpose**: Corporate fraud and misconduct investigations
- **Structure**: Corporate governance and compliance focus
- **Includes**: Entity mapping, financial analysis, regulatory frameworks

### Evidence Management Template
- **Purpose**: Digital and physical evidence handling
- **Structure**: Chain of custody and integrity verification
- **Includes**: Evidence logs, verification procedures, integrity checks

## Using Templates

1. Copy the appropriate template directory to `../active/`
2. Rename to follow case naming convention: `case_YYYY_XXXXXX`
3. Update template placeholders with case-specific information
4. Begin analysis using framework tools

## Template Structure

Each template includes:
- `README.md` - Case overview and instructions
- `timeline.md` - Timeline framework
- `evidence/` - Evidence management structure
- `analysis/` - Analysis worksheets and tools
- `reports/` - Report templates and outputs

Templates are designed to work with the HyperGNN Framework and other analysis tools in this repository.
