## Technical Notes

### Diagram Format: Mermaid

All diagrams use Mermaid syntax for:
- **Portability:** Renders in GitHub, markdown viewers, documentation tools
- **Version Control:** Text-based, easy to track changes
- **Accessibility:** Can be converted to SVG, PNG, or PDF for court
- **Maintainability:** Easy to update and modify

### Rendering Instructions

**GitHub:** Diagrams auto-render in GitHub markdown viewer  
**Local:** Use Mermaid preview extension in VS Code, Obsidian, or other markdown editors  
**Court Documents:** Export to PDF using mermaid-cli or online tools  
**Presentations:** Export to PNG/SVG for PowerPoint/Keynote

### Color Coding Legend

- **Green (#90EE90):** Legitimate entities (RegimA Zone Ltd, Daniel, Jacqueline)
- **Blue (#87CEEB):** Platform infrastructure and services
- **Red (#FF6347, #FF0000):** Problematic entities/actions (RWD appropriation, Rynette control, Peter)
- **Gold (#FFD700):** Intermediary/questionable actions
- **Light Pink (#FFB6C1):** Deficiencies/absences (RWD's lack of assets)
