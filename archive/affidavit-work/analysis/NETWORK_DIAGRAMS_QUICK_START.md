## Technical Notes

### Mermaid Format
- **Text-based:** Easy to version control and modify
- **Auto-render:** GitHub, many markdown viewers render automatically
- **Export:** Can convert to PNG, SVG, PDF for court documents
- **Accessible:** Can be described verbally for accessibility

### Color Coding
- **Green (#90EE90):** Legitimate entities (RegimA Zone Ltd, Daniel, Jacqueline)
- **Blue (#87CEEB):** Platform infrastructure and services
- **Red (#FF6347, #FF0000):** Problematic entities/actions (RWD appropriation, Rynette control)
- **Gold (#FFD700):** Intermediary/questionable (Peter as trustee)
- **Light Pink (#FFB6C1):** Deficiencies/absences (RWD's lack of assets)
