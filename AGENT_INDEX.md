## Using This Index

### For Legal Team
1. Identify which case and claim you're working on
2. Review primary evidence list for that claim
3. Navigate to specific annexure locations
4. Cross-reference with ARENA_INDEX.md for applicable legal principles
5. Use case-specific agent-mapping.md for detailed connections

### For Evidence Collection
1. Use this index to identify missing evidence
2. Check evidence status (available, pending, blocked)
3. Plan evidence gathering based on priority levels
4. Track completeness metrics by case

### For Court Preparation
1. Assemble primary evidence for relevant claims
2. Organize supporting evidence in logical order
3. Prepare annexure bundles per case
4. Cross-reference with legal arguments (ARENA_INDEX.md)

### For Grip Optimization
1. Use GRIP_WORKFLOW.md for evidence analysis workflows
2. Run `npm run db:hypergraph:populate` to map evidence relationships
3. Run `npm run validate-evidence-completeness` to check coverage
4. Use `npm run db:hierarchy:stats` to see evidence organization
