## PART E: SECURITY MEASURES

### 1. Technical Security Measures

#### 1.1 Access Controls
- Role-based access control (RBAC)
- Multi-factor authentication (MFA) for all users (implemented August 1, 2025)
- Password requirements: minimum 12 characters, complexity, 90-day expiry
- Access logging and monitoring

#### 1.2 Encryption
- Data at rest: AES-256 encryption
- Data in transit: TLS 1.3 (minimum)
- Payment data: Tokenization + encryption (enhanced September 5, 2025)
- Encryption key management: Hardware security modules (HSMs)

#### 1.3 Network Security
- Firewall: Next-generation firewall (NGFW) with intrusion prevention
- Network segmentation: Separate networks for production, development, admin
- VPN: Required for remote access, AES-256 encryption
- Intrusion detection: 24/7 monitoring, automated alerts

- Antivirus: Enterprise-grade, real-time scanning, daily updates
- Endpoint detection and response (EDR): Deployed to all endpoints

- Patch management: Automated patching, monthly compliance checks
